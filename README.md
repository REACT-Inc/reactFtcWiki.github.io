# What is REACT Inc.
REACT, or Robotics and Engineering in Allegany County--Together, Inc., is the local non-profit organization that provides Teams with a meeting- and working-space, additiv manufacturing opportunities, e.g. 3d printer, as well as other materials needed to build the teams ropbot to compete in their respective qualifiers.

Further information regarding REACT can be found [here](https://reactallegany.weebly.com/)

# What is 4H
We partnered with the 4H programm which is a youth develoment organization that helps youth develop and deepen their skills and consits of over 100 universities nation wide.

Further information about 4H can be found on their [website](https://4-h.org/)
