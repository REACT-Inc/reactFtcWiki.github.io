# How To's
## Learn about robots
The [FIRST documentation page](https://ftc-docs.firstinspires.org/en/latest/index.html) is the best initial place to look for information.
To learn more about the game, check out the latest season competition manual, found [here](https://ftc-resources.firstinspires.org/ftc/game). This should give you an overview of what to expect during a season.

## Start with coding
In case you don't know how to code yet, a basic online course can help grasp the concepts utilized by programming languages and pepares you for any further. This is an example for a basic course, no affiliation or endorsement, [Java tutorial](https://www.codecademy.com/learn/learn-java)

If you already posess the basic knowledge of Java or any other programming language, good place to start to learn is [FTCSIM](https://ftcsim.org/).
Here you can learn in an online environment with visual feedback, by controlling a robot through increasingly difficult tasks. Give it a try and see if it is for you. 

FIRST also has tutorials for [Blocks](https://ftc-docs.firstinspires.org/en/latest/programming_resources/blocks/Blocks-Tutorial.html), [OnBot Java](https://ftc-docs.firstinspires.org/en/latest/programming_resources/onbot_java/OnBot-Java-Tutorial.html) and [Android Studio](https://ftc-docs.firstinspires.org/en/latest/programming_resources/android_studio_java/Android-Studio-Tutorial.html) (native java) in their documents. Have a look and see, if these information help you out.

During practice the option for hands-on learning can be utilized with out testbench, consisting of a control hub, which controls a motor, two servo motors and other sensors and LEDs to learn how your code can affect the hardware.

## Start with CAD (Computer Aided Design)
For our CAD needs we are using [ONSHAPE](https://www.onshape.com/).

If you already have a login you can send an email to one of the coaches to be added to the onshape classroom. This way you can access all the past done designs of the team and jump right into creating a new robot.

No account yet? No problem just create one for free and let one of the coaches know to add you to the ONSHAPE classroom.

ONSHAPE also offers a tutorial which introduces you to the basic operations of the tool and shows how to find the right instruction to achieve the desired design.

There are other CAD programs that could be used but ONSHAPE lets us collaborate very easily on a design without the need for another tool.



Check out our [Teams CAD page](https://ftc-docs.firstinspires.org/en/latest/cad_resources/index.html?highlight=cad) or the [FIRST documentation](https://ftc-docs.firstinspires.org/en/latest/cad_resources/index.html?highlight=cad) about CAD
